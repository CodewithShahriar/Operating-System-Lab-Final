# 🖥️ Operating System Lab Final

## 📌 Overview
This repository contains implementations of various Operating System algorithms written in C.  
It focuses mainly on CPU Scheduling algorithms and Deadlock Avoidance (Banker’s Algorithm).

---

## 🎯 Objectives
- To understand CPU Scheduling algorithms
- To implement different scheduling techniques
- To simulate deadlock avoidance using Banker’s Algorithm
- To analyze process execution time and efficiency

---

## 📂 Files Included

| File Name     | Description |
|--------------|------------|
| fcfs.c       | First Come First Serve Scheduling |
| sjf.c        | Shortest Job First Scheduling |
| srtf.c       | Shortest Remaining Time First |
| lrtf.c       | Longest Remaining Time First |
| ljf.c        | Longest Job First |
| priority.c   | Priority Scheduling |
| banker.c     | Banker’s Algorithm (Deadlock Avoidance) |
| all.c        | Combined implementation |

---

## ⚙️ Requirements
- GCC Compiler
- Linux / Unix Environment

---

## ▶️ How to Run

Compile any file:
```bash
gcc filename.c -o output
./output
````

Example:

```bash
gcc fcfs.c -o fcfs
./fcfs
```
